Title Project:  Medico (Child vaccination tracking system)

Team: Neuro

Problem statement: In today's busy life parents are not able to track the vaccinations of their child which may risk the child's health in their child age.

Solution of problem:
For the above problem we have built a prototype web application which will solve it. This is a child vaccination tracking system.

What are the functionalities it consists:
Login,signup and logout to new user
Chat feature with ai for personal recommendation.
Child profile section
Premium feature accessibility

Overall architecture, tools and technologies used:

The project shares a single codebase in vscode, which we can also call three-tier monorepo architecture.
Frontend (React)
Backend (Nodejs and Express)
Database (MongodbAtlas a cloud service provider)
